# 简介

> 此项目是一个使用WePY写的一个小程序，算是一个练手小demo

## 项目运行

``` bash
1.进入项目根目录，安装依赖
npm install

2.启动编译
npm run dev

3.使用微信开发者工具将dist目录添加进去即可看到效果
```
